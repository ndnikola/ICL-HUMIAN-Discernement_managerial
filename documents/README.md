# Documents

Ce dossier n'est pas utilisé par la version actuelle de l'écran 11 (les documents fictifs y sont représentés par des icônes, pas par des fichiers réels), afin d'éviter toute dépendance à des fichiers manquants le jour de la séance.

Si des documents fictifs réels sont préparés (échange de mails, procédure, compte rendu, tableau de suivi, document en anglais, deux versions d'un document), ils peuvent être déposés ici et reliés depuis `data/contenus.js` (écran `ecran-11`, tableau `documents`).
